Course Description

Exploration of the fundamental operations and concepts of computer technologies to facilitate learning in today's P-12 classrooms.

Getting Started

  1. Read the course syllabus.
  2. Review "Assignments" and "Projects"
  3. Begin completing those assignments and projects
  4. Submit the completed assignments on Blackboard before the deadlines.

Course Learning Objectives

In this course, you will:

  1. Understand the basic elements of Microsoft Office
  2. Utilize Microsoft Word, Excel, and PowerPoint and their features
  3. Create documents, spreadsheets and PowerPoint presentations with an educational context
  4. Understand how to use Blackboard to submit assignments and access resources in an entirely online course
